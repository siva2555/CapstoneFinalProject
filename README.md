# Digital Maintenance Tracker for Apartments / Buildings

A web-based application to digitally manage apartment and building maintenance activities.

---

## Project Overview

The **Digital Maintenance Tracker** is designed to streamline communication and operations between residents, building administrators, and maintenance staff.  
It provides transparency, improves response time, and maintains secure digital records for all maintenance-related activities.

---

## Objectives

- Manage maintenance complaints
- Track maintenance payments
- Assign and monitor maintenance tasks
- Improve transparency between residents and management
- Maintain secure digital records

## System Workflow

1. Resident logs in and views maintenance details.
2. Complaints are raised through the application.
3. Admin assigns tasks to maintenance staff.
4. Staff updates the task status.
5. Payments are recorded and tracked.
